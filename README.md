# anoriqq/messi

[messi](https://messi.anoriqq.com) will randomly assemble a team.

## File format

```json:members.json
{
  "members": [
    {
      "name": "anoriqq",
      "icon": "https://example.com/icon-image-1.jpg"
    },
    {
      "name": "moqmoqq",
      "icon": "https://example.com/icon-image-2.jpg"
    }
  ]
}
```
